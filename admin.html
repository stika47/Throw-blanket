🔗 ربط Google Sheets

لربط الطلبات مع Google Sheets، اتبع الخطوات التالية:

  1. أنشئ جدول Google Sheet جديد
  2. اذهب إلى Extensions → Apps Script
  3. الصق كود Apps Script (اضغط الزر أسفله لنسخه)
  4. اضغط Deploy → New deployment — Type: Web app — Access: Anyone
  5. انسخ رابط Web App والصقه هنا
