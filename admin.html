ربط Supabase

بدل Google Sheets، نستعملو Supabase — قاعدة بيانات مجانية وسهلة:

  1. روح لـ supabase.com وسجل مجاناً
  2. أنشئ مشروع جديد (New Project)
  3. روح لـ SQL Editor والصق كود إنشاء الجدول (اضغط الزر أسفله)
  4. اضغط Run باش تتنفذ
  5. روح لـ Project Settings → API
  6. انسخ Project URL و anon public key والصقهم هنا
